# Bae Jinwoo GitHub Pages Portfolio

This folder is a static GitHub Pages-ready portfolio for `bgw4399.github.io`.

## Files

- `index.html`: portfolio page
- `styles.css`: responsive styling
- `assets/`: selected project and award visuals

## Deploy

1. Create a public repository named `bgw4399.github.io`.
2. Copy all files from this folder to the repository root.
3. Commit and push to the `main` branch.
4. Enable GitHub Pages if it is not automatically enabled.

Do not commit private datasets, tokens, Excel files with contacts/addresses, medical records, or large Kaggle model artifacts.
